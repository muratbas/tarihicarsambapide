# Tech Context

## Technologies Used
- Next.js 15 (App Router), TypeScript, Tailwind CSS v4
- Firebase (Firestore, Auth, Storage)
- Node.js & npm

## Design Choices
- Fonts: Lora (Headings/Buttons) & Outfit (Body Text)
- Colors: Primary #EA580C, Secondary #EA580C, Background #FAFAF9, Text #1c1917
- Mobile-first approach but fully responsive for desktop.
