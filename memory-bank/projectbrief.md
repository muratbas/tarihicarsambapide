# Project Brief

## Core Requirements & Goals
A modern, mobile-first QR menu and corporate website for a restaurant. 
The site will allow users to scan a QR code at their table to view the menu, see company information, and initiate a phone call for orders.
- Responsive design with a focus on mobile-first user experience.
- Dynamic menu system managed via an admin panel.
- Firebase used for Authentication and Firestore for the database.
- Multi-language (TR/ENG) support planned for the future.
- Typography: Lora (Headers, Buttons), Outfit (Text).
- Colors: Primary #EA580C, Secondary #EA580C, Background #FAFAF9, Text #1c1917.
